__URL__ 为当前模块url。
如我在Teacher的teacher下时，跳转同样在Teacher下的change_password，那么href="__URL__/change_password"

__ROOT__为本项目根目录，即/talent_pool/


__PUBLIC__为/talent_pool/Public/


现在暂时还是不要公用模板Public了，如果模板html一样的我们直接复制过去就好了。

像change_password和logout这样的
//=====JS CODE======
$("#logout").click(function(){
	window.location.href="__ROOT__/";		//登出直接到主页就好
});

$("#change_password").click(function(){
	window.location.href = "__URL__/change_password";
});
//=====END  JS CODE======
如上四个角色的模板基本都一样的js可以考虑放到Public/js里自己写个common.js
